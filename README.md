# dwm-bootstrap
Minimal dwm bootstrap for base Linux installations

## Components
- dwm
- dmenu
- slstatus

## Config Defaults
- Terminal: Alacritty
- Font: IosevkaTerm Nerd Font

## Patches

### dwm
vanitygaps, alwayscenter, attachaside, cyclelayouts, rotatestack

### dmenu
alpha, center, grid, gridnav
